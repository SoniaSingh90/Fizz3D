About Fizz3D

A mobile-first, responsive web application showcasing interactive 3D beverage models, developed as part of the Mobile Web 3D Applications module at the University of Sussex.

Project Overview

Fizz3D is an interactive web application that brings beverage products to life through 3D visualization. Using modern web technologies, we've created an immersive experience that works seamlessly across all devices.

Key Objectives
  • Real-time 3D rendering via the browser
  • Web-based interactivity using HTML, JavaScript, and X3D
  • Practical application of skills learned through structured labs
Interactive 3D Experience

Experience our beverage models in stunning 3D with interactive controls for rotation, camera angles, and animations.

Technologies Used

3D Technologies
  • Blender – 3D modeling and animation
  • Three.js – GLTF model rendering
  • X3DOM – X3D model integration
Web Technologies
  • HTML5 & CSS3 – Structure and styling
  • JavaScript – Interactivity and animations
  • PHP & AJAX – Dynamic content loading
UI Framework
  • Bootstrap 5 – Responsive layout
  • Font Awesome – Icon system
  • Custom CSS – Theme and animations
